PJCE's Progress Blog

Peter J C Evans

This site began as a record of self-study progression and a delving into GitHub Pages and Jekyll, but has since evolved to become a place for cataloguing small projects that I have worked on as well as reflecting on the process of learning different technologies.

This website is built with Jekyll the source code for the website is available in this GitHub repo.

Aside from Jekyll and the integrated Liquid templating language no other third party code, in the form of libraries or frameworks, has been included. This is intentional as desipte the easy facility that libraries like Bootstrap or jQuery offer, their use seperates developers from the underlying code and can force people to think within a box, which an aspiring developer in a rapidly evolving field cannot afford to do.

PJCE: Developer

Peter J C Evans

This site began as a record of study progression - focussed initially on delving into GitHub Pages and Jekyll. It has since evolved to include a catalogue of small projects that I have worked on as well as a blog which reflects on the process of learning different technologies.

This website uses Jekyll to logically separate elements of the websiteâ€™s code in order to make it easily manageable. The source is available in this GitHub repo.

Aside from Jekyll and the integrated Liquid templating language no other third party code, in the form of libraries or frameworks, has been included. This is intentional as despite the easy facility that libraries, such as Bootstrap or jQuery, offer their use separates developers from the underlying code. I believe this can be restricting as it leads to thinking within a limited box, which an aspiring developer in a rapidly evolving field can rarely afford to do.
